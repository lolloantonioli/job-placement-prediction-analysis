# Job Placement Prediction Analysis

This project analyzes the effectiveness of various job search channels and utilizes data analysis and machine learning techniques to explore the key variables influencing candidates' professional placement.

## Repository Structure

* **`job_board_effectiveness.csv`**: The dataset used for the analysis, containing records of different recruitment platforms and candidate placement outcomes.
* **`job_placement_prediction_analysis.ipynb`**: The main Jupyter Notebook containing Exploratory Data Analysis (EDA), data preprocessing, visualization, and predictive model training.

## Technologies and Tools
* **Python**
* **Jupyter Notebook**
* **Data Science Libraries**: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
